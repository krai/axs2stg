# axs2stg

Automated [KRAI X](https://github.com/krai/axs) workflows for running Stable Diffusion XL with the [StageAI](https://app.thestage.ai/) backend using the LoadGen client.

To import this repository and its dependencies into your **work_collection**, run:
```
axs byquery git_repo,collection,repo_name=axs2stg
```
